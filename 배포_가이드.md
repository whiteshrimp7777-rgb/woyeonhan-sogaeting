## 3. CLI 도구 경로 (중요)

이 머신에서는 기본 셸 PATH에 Homebrew 경로가 포함되지 않을 수 있습니다.
CLI 도구를 사용하기 전에 **반드시 PATH를 설정**하거나, **절대 경로로 실행**해야 합니다.

### 설치 확인된 도구 및 절대 경로

| 도구 | 절대 경로 | 비고 |
|------|-----------|------|
| Homebrew | `/opt/homebrew/bin/brew` | macOS Apple Silicon |
| Node.js | `/opt/homebrew/bin/node` | Homebrew로 설치됨 |
| npm | `/opt/homebrew/bin/npm` | Homebrew로 설치됨 |
| Git | `/usr/bin/git` | macOS 기본 내장 |
| Netlify CLI | 미설치 | 아래 설치 방법 참고 |

### PATH 설정 방법

명령어 실행 전에 아래를 먼저 실행하면 `node`, `npm`, `git` 등을 이름만으로 사용 가능:

```bash
export PATH="/opt/homebrew/bin:$PATH"
```

또는 절대 경로로 직접 실행:

```bash
/opt/homebrew/bin/node --version
/opt/homebrew/bin/npm --version
```

### PATH가 안 잡힐 때 경로 찾는 법

```bash
# Homebrew 기반 도구 찾기 (Apple Silicon Mac)
ls /opt/homebrew/bin/

# Homebrew 기반 도구 찾기 (Intel Mac)
ls /usr/local/bin/

# 특정 도구 찾기
find /opt/homebrew /usr/local -name "netlify" -type f 2>/dev/null
find /opt/homebrew /usr/local -name "node" -type f 2>/dev/null

# npm 글로벌 패키지 경로 확인
/opt/homebrew/bin/npm root -g
# → /opt/homebrew/lib/node_modules

# npm 글로벌 설치된 패키지 목록
/opt/homebrew/bin/npm list -g --depth=0
```

## 4. Netlify CLI 설치 (미설치 상태)

현재 이 머신에 Netlify CLI가 설치되어 있지 않습니다. 설치 방법:

```bash
export PATH="/opt/homebrew/bin:$PATH"
npm install -g netlify-cli
```

설치 후 경로 확인:
```bash
which netlify
# 예상 경로: /opt/homebrew/bin/netlify
```

최초 1회 로그인 필요:
```bash
netlify login
# 브라우저가 열리면 Netlify 계정으로 로그인
```

## 6. 배포 방법

### 방법 A: Netlify CLI 직접 배포 (Netlify CLI 설치 필요)

```bash
# PATH 설정
export PATH="/opt/homebrew/bin:$PATH"

# 프로젝트 폴더 이동
cd "/Users/home366866/Desktop/<우연한 모음집>/우연한 사이트"

# 테스트 배포 (Draft URL 생성)
netlify deploy

# 실제 배포 (프로덕션 반영)
netlify deploy --prod
```

### 방법 B: Git → GitHub Push → Netlify 자동 배포

```bash
# PATH 설정
export PATH="/opt/homebrew/bin:$PATH"

# 프로젝트 폴더 이동
cd "/Users/home366866/Desktop/<우연한 모음집>/우연한 사이트"

# 변경사항 확인
git status

# 파일 스테이징
git add 수정한파일.html
# 또는 전체: git add .

# 커밋
git commit -m "변경 설명"

# GitHub에 푸시 → Netlify 자동 배포
git push origin main
```

## 9. 문제 해결

### "command not found" 오류
PATH가 안 잡혀 있는 경우. 아래를 먼저 실행:
```bash
export PATH="/opt/homebrew/bin:$PATH"
```

### Git push 거부
원격에 변경사항이 있을 때:
```bash
git pull origin main
git push origin main
```

### 배포 후 변경이 안 보일 때
- `Cmd + Shift + R`로 브라우저 캐시 비우기
- 시크릿/프라이빗 모드에서 확인
- `netlify open`으로 Netlify 대시보드에서 배포 상태 확인
